# Modul Code & Design, HFIAD25
Dieses Projekt ist Teil des Moduls Code & Design der Klasse HFIAD25 an der Schule für Gestaltung Zürich, betreut von Hanna Züllig.


## Aufgabe 1 – Slider: Beziehungen steuern 
Die p5.js-Skizze «Harmony & Chaos» visualisiert das Zusammenspiel von Ordnung und Zufall.
Zwei Slider steuern das Verhalten von vier transparenten, farbigen Kreisen:
– Der Growth-Slider bestimmt die Wachstums-Geschwindigkeit.
– Der Chaos-Slider regelt die Unterschiede im Wachstum der einzelnen Kreise.

Trotz zunehmendem Chaos entstehen durch Überlagerungen und Transparenzen neue harmonische Farbkompositionen – ein visuelles Spiel zwischen Struktur und Unvorhersehbarkeit.
